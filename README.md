## 1. プロジェクト概要
カードゲームの管理者向けダッシュボードおよびユーザー向けカード表示システムがメインです。  
管理者はカード・ガチャ・コインの管理、ユーザーへの付与を行えます。  
ユーザーはカードの確認・強化・進化・お気に入り登録、そしてガチャが行えます

---

## 2. 主な機能

### 管理者画面
- カード作成
- ユーザーへのカード付与
- ユーザーへのコイン付与
- ガチャ作成 / 編集 / 削除
- ガチャ利用履歴確認


### ユーザー画面
- コインの獲得
- カード一覧表示（ホログラムアニメーション）
- カード詳細モーダル表示（ステータス・レアリティ表示）
- お気に入り登録 / 強化 / 進化
- 素材カード選択モード対応
- ガチャ

---

## 3. Docker 環境構築手順

### 環境の立ち上げ方

`docker compose up -d --build`

initialize.sqlで基本データを挿入しているのでSQLの操作は不要です。

### Webブラウザからアクセス

ユーザー画面:`http://localhost/login.php`

管理者ページ: `http://localhost/admin/admin_login.php`

管理者ページへのログインパスワードは`admin123`です。
